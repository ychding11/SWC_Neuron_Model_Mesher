# SWC_Mesher
It is an independent clone of [swc_mesher](https://github.com/mcellteam/swc_mesher).
They have the same functionality. The original [swc_mesher](https://github.com/mcellteam/swc_mesher) only support Blender 2.7x.
This one upgrade it to Blender 2.80 and later.
As for usage & document you can visit readme in original repo [swc_mesher](https://github.com/mcellteam/swc_mesher).
